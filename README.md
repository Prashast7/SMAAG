# SMAAG

Production-grade Next.js 15 foundation for a premium CPA firm website.

## Stack

- Next.js 15 App Router
- TypeScript
- Tailwind CSS v4
- shadcn/ui-ready configuration
- ESLint + Prettier
- Husky + lint-staged
- pnpm

## Commands

```bash
pnpm dev
pnpm lint
pnpm typecheck
pnpm format
pnpm build
pnpm validate
```

## Structure

```text
app/
components/
constants/
hooks/
lib/
public/
styles/
types/
```

## Environment

Copy `.env.example` to `.env.local` and update values before deployment.
